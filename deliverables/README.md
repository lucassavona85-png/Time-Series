# Deliverables / Livrables

Ce dossier contient les livrables finaux et documents du projet.

## Contenu

### Rapports PDF
Placez ici les rapports finaux :
- Rapport d'analyse
- Documentation technique
- Synthèse des résultats

### Présentations
Placez ici les présentations :
- PowerPoint (.pptx)
- PDF des présentations
- Supports visuels

### Documents de Synthèse
Placez ici :
- Résumés exécutifs
- Fiches récapitulatives
- Documentation finale

## Conventions de Nommage
- Utilisez des noms clairs et datés
- Format recommandé : `YYYY-MM-DD_titre_document.pdf`
- Exemple : `2024-12-30_rapport_final_series_temporelles.pdf`

## Versions
- Conservez les versions importantes
- Utilisez un suffixe de version si nécessaire (v1, v2, etc.)
- Exemple : `presentation_finale_v2.pdf`
